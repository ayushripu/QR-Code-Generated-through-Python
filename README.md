## 📖 Overview

QR (Quick Response) codes are two-dimensional barcodes that can store URLs, text, contact information, and more. This project provides a simple way to generate QR codes programmatically using Python.

## ✨ Features

- Generate QR codes for any URL or text
- Save QR codes as PNG images
- Customizable size and error correction
- Fast and lightweight
- Easy to integrate into larger projects
